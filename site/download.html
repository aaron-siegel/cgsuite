Download CGSuite

 

To download CGSuite, choose one of the versions below. Most users will want the latest stable version, but if you're feeling adventurous and want access to all the newest features, you can try the latest beta instead.

Latest Beta Release:
All Earlier Releases: Github Releases Page


From the Release page, download the appropriate installer for your operating system (.dmg for Mac, .exe for Windows, or .sh for Linux), run it, and you're good to go!

Once you've installed CGSuite, you can access its extensive documentation in the Help system. If you're new to CGSuite, or if this is your first time using version 2.0 or later, a good place to start is the Getting Started tutorial (select "Getting Started" from the Help menu).
